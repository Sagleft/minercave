# minercave (WIP)
Multi-cryptocurrency CPU miner.

## How to start

1. Copy `config.example.json` to `config.json` and fill it.
2. Run miner: `./minercave`

## How to build from source

```bash
git clone https://github.com/Sagleft/minercave && cd minercave
go build
```
